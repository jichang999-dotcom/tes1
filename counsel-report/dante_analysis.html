핵심 이론 요약

밥그릇 기법 (세력 공략 프레임)

  • 1구간: 악재 → 급락 → 개미 투매
  • 2구간: 저가 매집 (1구간 대비 1:1 이상 시간)
  • 3구간: 224일선 돌파 → 본격 상승 (공략 포인트)
  • 4구간: 급등 → 호재 뉴스 동반

적용: 소형주 + 대주주 지분 20~40%

공구리 기법 (바닥 돌파)

  • 공구리 1: 전고점 매물 소화 → 지지 → 상승
  • 공구리 2: 쌍바닥 → 추가 공구리 → 상승
  • 공구리 3: 개미털기 후 상승 (가장 강력)

공구리 라인 이탈 시 즉시 손절

하이힐 / 256 / 이평때리기

  • 하이힐: 낙폭 과대 후 "굽" 형성 시 매수 타점
  • 256 기법: 224일선 터치 → 반등 구간 공략
  • 이평때리기: 주요 이평선 지지 확인 후 진입

224일선 = 세력 최중시 지지/저항

비중 배분 전략 (사용자 고정)

  • 스윙: 1:1:2 (3분할) → 25/25/50%
  • 중장기: 1:1:2:4 (4분할) → 12.5/12.5/25/50%
  • 물타기 구조: 지지선 + 반등 양봉 + 거래량 0.7~1.5배
  • 손절은 기계적, 수익은 추세에 맡김
