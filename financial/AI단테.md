## 1. 채널 개요

**주식단테사단**은 주식단테를 중심으로 한 트레이더 커뮤니티. 주요 강사진:

| 강사 | 전문 분야 |
|---|---|
| 주식단테 | 시초가 / 역매공파 / 밥그릇 — 전포지션 종합 |
| 주식바바 | 역매공파 / 공구리 / 눌림목 (중기 스윙) |
| 주식등대 | 밥그릇 / 엘리엇파동 (중장기) |
| 또사이선생 | 256 기법 / 오돌이 / 농사매매 / 미국주식 |
| 주식코주부 | 하이힐 / 파동에너지 (장중 단타) |

## 2. 핵심 이론 5가지

### 2.1 밥그릇 기법 (가장 핵심, 세력 공략 프레임)

세력이 종목을 공략하는 4단계 구조. 차트 모양이 밥그릇을 닮았다고 해서 붙여진 이름.

| 구간 | 의미 | 특징 |
|---|---|---|
| **1구간** | 바닥 형성 | 악재 유포 → 급락 → 개미 투매 유도 |
| **2구간** | 매집 | 저가 박스권. 1구간 대비 최소 **1:1 이상** 시간 소요 |
| **3구간** | 출발 (공략 포인트) | **224일선 돌파** → 본격 상승 시작. 개미털기 발생 |
| **4구간** | 급등 | 원래 가격 회복. 호재 뉴스 동반 |

**적용 대상**: 소형주 + 대주주 지분율 20~40% 종목에서 주로 발생.

**공략 타점**: 3구간 돌파 + 3구간 눌림목.

### 2.2 공구리 기법 (바닥 다지기 돌파)

"바닥을 잡는 게 아니라, **바닥을 만들고 올라오는** 구간" 공략. 단기 1~2주 추세용.

| 패턴 | 구조 |
|---|---|
| **공구리 1** | 전고점 부근 매물 소화 → 지지 형성 → 상승 |
| **공구리 2** | 쌍바닥 형태로 작은 공구리 추가 → 상승 |
| **공구리 3** | 전고점·다음 고점 사이에서 개미털기 후 상승 ⭐ **가장 강력** |

**규칙**:
- 224일선 아래 하락 추세 → 하락 박스 돌파 시 진입 신호
- 직전 언덕(전고점)이 지지선
- **공구리 라인 이탈 시 즉시 손절**

### 2.3 하이힐 기법

낙폭 과대 후 회복 과정에서 "굽"이 생기는 패턴.
- 굽이 생기면 → **하이힐** (매수 타점)
- 굽 없이 밋밋 → **밥그릇**으로 해석

### 2.4 256 기법 / 이평때리기

- 주가가 주요 이평선(특히 **224일선**) 근접·터치 후 반등하는 구간 공략
- 224일선 = 세력이 가장 중시하는 저항/지지선
- 224일선을 자주 오간 종목은 **골파기(개미 털기)** 확률 높음 → 주의

### 2.5 112/224/448 기법 (데드 기법 + 장기 이평 릴레이)

장기 이평 **112(녹)·224(검)·448(빨)** 을 활용한 2가지 기법.

#### 2.5.1 데드 기법 (부채꼴 수렴)

주가 급등 후 장기 이평이 부채꼴로 벌어졌다가 **다시 닫힐 때** 진입.

| 조건 | 내용 |
|---|---|
| 선행 조건 | 주가 **50% 이상 급등**, 파동 2~3번 이상 형성 |
| 진입 트리거 | 112/224/448선 중 하나에 주가가 **닿을 때 (반등 양봉)** |
| 반등 확률 | **약 90%** (커뮤니티 주장) |
| 예외 (진입 금지) | 엘리엇 3파 / 쌍봉 / 헤드앤숄더 / 이미 여러 번 닿은 이평 |

#### 2.5.2 장기 이평 릴레이 기법

큰 하락 후 **112·224·448 역배열 + 이격 확대** 상태에서 돌파 릴레이:

- **112일선 돌파** → 다음 저항 **224일선**까지 기대
- **224일선 돌파** → 다음 저항 **448일선**까지 기대
- 타점 실수로 물려도 시간투자로 본전 회복 가능 → **초보용 안정 기법**

**진입 룰**:
- ✅ 112/224일선 **종가 돌파 + 양봉 + 거래량 동반**
- ✅ 수렴 상태(112·224 밀집)에서 돌파 시도 양봉은 강한 신호
- ❌ 음봉으로 이평 돌파는 가짜 돌파 — 다음 날 재확인 필요
- 손절: 돌파한 이평 **종가 이탈 시 기계적 컷**

### 2.6 손절 3원칙

- 밥그릇·공구리 라인 **이탈 시 무조건 손절** (기계적)
- 수익보다 **계좌 방어**가 우선
- 손절은 룰, 수익은 추세에 맡김

## 4. 차트 분석 요청 프로토콜

사용자가 차트를 제공할 때 아래 절차로 분석한다.

### 4.1 사용자가 제공해야 할 정보

1. **차트 이미지** (가능하면 일봉 + 주봉 2장)
2. **종목명**
3. **매매 타입** (필수): **스윙** / **중장기** 중 하나 명시
   - 스윙: 3분할 매수 (1:1:2 비중)
   - 중장기: 4분할 매수 (1:1:2:4 비중)
4. **이평선 표시 필수**: 5 / 20 / 60 / 120 / **224일선**
5. **거래량 창 포함**
6. (선택) 현재가, 최근 주요 고점/저점, 대주주 지분율, 이미 체결된 매수가
7. **차트에 224일선이 없을 경우** 분석 정확도 저하 → 사용자에게 고지

**표준 요청 형식**:
```
차트: [이미지]
종목명: [종목명]
매매 타입: [스윙 / 중장기]
```

**⭐ 후보 → 활성 이동 시 후보 순위 재번호 룰 (2026-04-29 사용자 확정)**

후보 그룹에서 1종 이상이 활성으로 이동하면, **후보 탭에 남은 종목들의 #N 순위를 1번부터 다시 매긴다**. 활성 이동된 종목 자리는 비워두지 말고 후속 종목들이 위로 올라와 빈 번호가 없게 한다.

| 적용 위치 | 갱신 항목 |
|---|---|
| 후보 탭 종목 카드 헤더 | `🚦 #N` 텍스트 (예: 펄어비스 #3 → #1) |
| 후보 탭 카드 등장 순서 | DOM 상의 카드 순서를 새 #1 → #N으로 정렬 |
| 비교표 (후보 vs 한화솔루션) | "순위" 열의 #N 갱신 |
| 순위 요약표 (🚦 진입 유력도 순위) | "순위" 열의 #N 갱신 |
| 후보 캡션 (`🚦 후보 N종 운영 ...`) | 종목 수 + 카드 정렬 범위 갱신 |

**예시 (2026-04-29 적용)**: 지앤씨에너지(#1), 클래시스(#2)가 활성 이동 → 펄어비스(#3→**#1**), SKC(#4→**#2**), 보령(#5→**#3**), 금호타이어(#6→**#4**), 엔젤로보틱스(신규→**#5**)로 일괄 재번호.

**🚫 금지**:
- ❌ 활성 이동된 종목의 자리에 안내 highlight 박스 삽입 금지(영구 삭제 리스트 #5 항목과 중복)
- ❌ 후보 자리만 비워두고 #N은 그대로(예: #1, #2 비고 #3-#6 유지) 금지 — 빈 번호로 인한 혼동 차단

| 4 | **손절라인** | 손절가 (빨강 강조) |

**⭐ 진입 트리거 체크 표준 (2026-04-27 확정 — 모든 종목 공통)**

요약 배너 4종 바로 아래에는 **진입 트리거 체크 박스**를 배치한다. 활성·후보·중장기·단타 모든 종목 카드에서 **동일 형식**으로 운영해, 매일 시점별 체결 가능 여부를 한눈에 판정한다.

> 운영 원칙: "1차 진입 완료" 류의 서술형 highlight 박스 + "잔여 N주 추가 진입 룰" 류의 별도 warn-box는 **사용 금지**. 모든 진행 상태·분기 룰·체결 조건은 진입 트리거 체크 박스 안에서 표/하단 캡션으로 통합 표현한다.

| 항목 | 사양 |
|---|---|
| 박스 스타일 | `class="warn-box"` + `background:#1a2838; border-left:4px solid #58a6ff; margin-bottom:12px;` (파란 계열 고정) |
| 헤더 | `🚦 진입 트리거 체크 ([날짜] · [추가 컨텍스트])` — 추가 컨텍스트는 "잔여 1차 N주", "2차 진입", "박스 돌파 대기" 등 차수/모드 명시 |
| 표 헤더 | `# / 조건 / 기준 / [날짜] 현재 / 판정` 5열 고정 |
| 트리거 항목 | 4~6개 권장 (가격대·캔들·거래량·체결강도·이평선/박스 위치 + 종목별 갭/이벤트 조건) |
| 판정 표기 | ✅ 충족(녹색 #3fb950) / ⏳ 대기·확인(노랑 #d29922) / ⚠️ 약함(노랑 #d29922) / ❌ 미달(빨강 #f85149) |
| 표 하단 1번째 줄 | `🚨` 진입 종합 판정 (충족 시 녹색, 미충족 시 빨강/노랑). 체결 완료분·대기 차수·총 비중 구조 명시 |
| 표 하단 2번째 줄 | `🟡 분기 룰` — 갭상/갭하·트레일링·Track 전환 등 시나리오 분기 |
| 표 하단 3번째 줄 (선택) | `📐 이평선` 또는 `📊 거래량 추이` 등 보조 컨텍스트 (회색 #8b949e, 0.85em) |

**활성 종목 적용 룰**: 이미 체결된 차수가 있어도 박스 형식은 동일하게 유지한다. 체결 완료분은 표 안에서 "✅ 충족 ([차수] [수량] 체결)"로 표기하고, 대기 중 차수의 트리거를 함께 추적한다. 체결 이력·이평선 정보는 박스 하단 캡션 또는 단계별 통합 테이블의 매수일자 열로 흡수한다.

**⭐ 텍스트 요약 블록 사용 금지 (2026-04-27 확정 — 모든 종목/탭 공통)**

대시보드 상단·각 탭 상단·종목 카드 어디에서도 **불릿/문장 나열식 prose 요약 블록 사용 금지**. 시세·운영현황·진입유력도·액션플랜은 반드시 **표(table)** 또는 **카드 내 trigger 박스** 형식으로 구조화한다.

> 사용자 확정 (2026-04-27): "글로 보여주는건 하지 않도록" — prose 요약은 정보가 분산되고 시점별 비교·판정이 어려움. 표 형식은 한눈에 충족/미달 판정 + 카드 간 일관 비교 가능.

| 금지 (prose) | 권장 (table) |
|---|---|
| ❌ "📊 2026-XX-XX 시세 (활성 N종 + 후보 N종)" 다음에 종목별 불릿 나열 | ✅ `<table>` 컬럼: 구분 / 순위 / 종목 / 현재가 / 변동 / 트리거 / 다음 액션 |
| ❌ "🚦 진입 유력도 순위" 다음에 #1 #2 #3 ... 문단 나열 | ✅ `<table>` 컬럼: 순위 / 종목 / 상태 / 핵심 신호 / 다음 액션 / 진입가 / 손익비 |
| ❌ "🚨 액션 플랜 — ① ... ② ... ③ ..." 형식 prose | ✅ 각 종목 카드 내부 진입 트리거 체크 박스의 표 + 분기 룰 캡션 |
| ❌ "📌 활성 포지션 (N종): A ... B ..." 산문 | ✅ 운영 현황 표의 "구분" 열에 활성/후보/종결 분류 |

**예외**: 14자 이내 보조 캡션 (예: 회색 작은 글씨 한 줄 안내), 비교표·순위표 직전 한 문장 설명, 카드 내부 trigger 박스 하단 🚨/🟡/📐 캡션 3줄은 허용. **여러 종목을 나열하는 산문 단락은 금지**.

**적용 시점**: 2026-04-27부터 모든 신규/갱신 분석에 적용. 기존 prose 블록은 갱신 시 표로 교체.

- **총 투자금 200만원**을 종목 수로 균등 배분 (예: 3종목 → 종목당 ~67만원)
- 스윙 1:1:2 → 종목 투자금의 **25% / 25% / 50%** 분할
- 중장기 1:1:2:4 → 종목 투자금의 **12.5% / 12.5% / 25% / 50%** 분할
- 이익금 = 누적 투자금 × 이익률
- 사용자가 요청 시 `투자금: [금액]` 명시하면 해당 금액 사용

3. 손익비 & 리스크 노트
   - 전량 체결 시 손익비 최소 1.5 이상 목표
   - 1차만 체결 시 손익비가 낮으므로 경고 명시
   - 소형주/유동성 낮은 종목은 실제 비중 축소 권장

### 4.3 비중 배분 — 매매 타입별 고정 룰 ⭐

사용자 표준 전략은 매매 타입에 따라 **고정**되어 있다.

#### 🟦 스윙 (기본) — **1:1:2 (3분할)**

| 차수 | 비중 | 성격 |
|---|---|---|
| 1차 | **25%** (1/4) | 탐색 진입 |
| 2차 | **25%** (1/4) | 지지 확인 |
| 3차 | **50%** (2/4) ⭐ | 최종 방어선 + 메인 포지션 |

- 평단 효과가 크고 손익비 유리
- 3차가 체결되어야 진가 발휘
- 1~2차만 체결되고 반등 시 수익은 제한적

#### 🟥 중장기 — **1:1:2:4 (4분할)**

| 차수 | 비중 | 성격 |
|---|---|---|
| 1차 | **12.5%** (1/8) | 탐색 진입 |
| 2차 | **12.5%** (1/8) | 지지 1차 확인 |
| 3차 | **25%** (2/8) | 지지 2차 확인 |
| 4차 | **50%** (4/8) ⭐ | 최저가 방어선 + 메인 포지션 |

- 가장 공격적인 평단 조정
- 진입 구간이 넓어 시간 여유 필요 → 중장기 전용
- 4차가 체결되지 않으면 효과 반감

#### 🆕 1차 매수 2분할 원칙 ⭐⭐ (2026-04-28 확정 / 2026-04-29 강조 재확인 — 모든 종목 공통, 스윙·중장기·후보·신규)

**1차 매수는 항상 2분할(1차a / 1차b)** 로 진행한다 = "1차 시작가를 2번 나누어 진입". 시간 분산 + 갭리스크·슬리피지 회피 + 트리거 재확인 목적.

> **사용자 표현 (2026-04-29)**: "1:1:2 비중에서 1차 시작가를 2번 나누어 진입한다." — 즉 비중 룰(스윙 1:1:2 / 중장기 1:1:2:4)의 1차분을 다시 1차a/1차b 두 시점·두 가격으로 분산한다는 명시적 의미. 모든 신규/갱신 종목 카드에 예외 없이 적용.

| 차수 | 비중 | 진입 시점 | 매수 조건 | 1차a 대비 가격대 |
|---|---|---|---|---|
| **1차a** | 1차 비중의 **50%** | 첫 트리거 충족 시 (어닝 후 첫 양봉, 박스 돌파 후 눌림양봉 등) | 양봉 + 체강 100+ + 거래량 평상시 (0.7~1.5배) | 기준 (현재가/돌파가/지지 양봉가) |
| **1차b** | 1차 비중의 **50%** | 1차a 후 **1~3영업일 內** 추가 눌림 또는 박스 재지지 | 양봉 + 1차a 가격 **±2% 범위 (눌림 우선)** ⭐ | 1차a 대비 **-1~-2% 권장** (하한 -2% 룰 위반 시 재산정) |

**🚨 ±2% 룰 엄수 (2026-04-29 명시)**: 1차a/1차b 가격 격차가 -2%를 초과하면 **1차b는 사실상 2차 영역**이 되어 1차 통합 평단·손익비 산출이 왜곡된다. 1차b가 1차a 대비 -2% 이내가 되도록 **매수가 재산정**하거나, 격차가 큰 가격은 2차로 재분류한다.

**비중 환산**:
- 🟦 스윙 1:1:2 → 1차 25% = **1차a 12.5% + 1차b 12.5%** + 2차 25% + 3차 50%
- 🟥 중장기 1:1:2:4 → 1차 12.5% = **1차a 6.25% + 1차b 6.25%** + 2차 12.5% + 3차 25% + 4차 50%

**자동 분기 룰** (1차a 체결 직후 갭 발생 시):
- ✅ **갭상 +2% 이상**: 1차b 자동 취소 → 1차a 단독 체결 종료, **트레일링 모드** 전환
- ✅ **갭하 -3% 이상**: 1차b 자동 취소 → **Track C(후퇴 모드)** 발효, 2차 가격대로 재대기
- ✅ 갭 ±2% 이내 보합: 1차b 정상 진행 (눌림 양봉 시 진입)

**🚫 금지 사항**:
- ❌ 1차 일괄 진입 금지 (단일 시점 슬리피지·갭리스크 노출)
- ❌ 1차a 음봉 진입 금지 (예측 매수)
- ❌ 1차b를 1차a보다 +2% 이상 비싸게 추격 매수 금지 (평단 상승)

**예외 (3분할 1차)**:
- 박스 중간 탐색 진입(시프트업 04-27 케이스) 등 명확한 트리거 부재 상황에서는 1차a/b/c **3분할 운용 가능**
- 활성 종목에서 사용자 확정으로 1차 잔여분 추가 진입을 분리 운용 시 (컨텍 04-27 케이스: 1차a 6주 체결 + 잔여 1차b 6주 대기) 동일 룰 적용

**적용 범위**: 2026-04-28부터 모든 신규/갱신 분석에 적용. 기존 종목 카드는 갱신 시 1차 → 1차a/1차b로 분할 재구성.

### 4.4 분할매수 / 목표가 / 손절 산출 룰

**매수 타점 선정 원칙 (공통)**
- **1차**: 현재 패턴의 1차 지지선 (예: 공구리 라인, 20일선 근처)
- **2차**: 2차 지지선 (직전 저점, 60일선)
- **3차 (마지막)**: 최종 방어선 (손절 직전, 밥그릇 2구간 하단, 224일선 근처)
- **4차 (중장기만)**: 장기 박스권 하단 또는 밥그릇 2구간 최저점

**매수 신호 조건 — 눌림 매수 vs 돌파 매수 구분 필수** ⭐⭐

| 구분 | 상황 | 조건 | 거래량 |
|---|---|---|---|
| 🟢 **눌림 매수** (물타기 포함) | 지지선 눌림 터치 후 반등 | **반등 양봉** (꼬리양봉 OK) | **평균의 0.7~1.5배** (평상시 수준) |
| 🔴 **돌파 매수** (3구간 출발) | 박스/저항선 종가 돌파 | **몸통양봉** (꼬리양봉 X) | **평균의 2배 이상** (핵심 조건) |

**🚨 혼동 금지 룰** (2026-04-23 사용자 확정):
- ❌ **눌림 매수에 "거래량 2배" 조건 적용 금지** — 돌파 매수 기준을 잘못 쓰는 것. 눌림은 원래 거래량 줄어드는 게 정상
- ❌ 음봉 진입 금지 — 예측 매수는 칼날 잡기
- ❌ 눌림에서 거래량 2배 터지면 오히려 **투매 신호** 가능 → 반등 확인 후 진입
- ✅ 한 종목에서 1차는 돌파 매수 / 2~3차는 눌림 매수처럼 **차수별로 다른 룰 적용 가능**
- ✅ 현재가가 지지선과 사실상 같은 수준(-1% 이내)이면 양봉 확인 후 **지정가 진입**으로 충분

**🚨 박스 돌파 매수 — 1차 매수가 위치 룰** ⭐ (2026-04-29 사용자 확정):
- ✅ **박스 돌파 매수 시 1차a/1차b는 모두 박스 상단 위(돌파선 위) 가격에 위치**
  - 예: 박스 상단 30,000원 → 1차a 31,000 (박스 +3.3%, 돌파 종가 양봉 안착) / 1차b 30,500 (박스 +1.7%, 박스 상단 재테스트 양봉)
  - 1차a/1차b 격차는 ±2% 룰 충족 (4.3절)
- ❌ **1차a 또는 1차b 가격이 박스 안쪽(돌파선 아래)에 위치하는 것 금지**
  - 가짜 돌파 시 박스 안쪽 가격은 자동 손실 영역 → 단테 룰 위반
  - 박스 안쪽 가격은 차수 의미가 "박스 돌파 1차"가 아닌 "박스 안 눌림 매수 1차"로 변질됨 (룰 일관성 훼손)
- ✅ **2차/3차는 박스 돌파 실패 시 가짜 돌파 시나리오에서 박스 안쪽 또는 깊은 눌림 영역**
  - 예: 2차 28,000 (박스 안 5/20일선 부근), 3차 25,500 (224일선 부근)
- ✅ **자동 분기 룰의 "갭하 -3%" 기준은 박스선 기준**으로 잡음
  - 1차a 체결 후 박스선 아래로 회귀 시 → 1차b 자동 취소 + Track C(가짜 돌파 확정 → 2차 영역 직진)
- 📐 **눌림 매수와의 차이**: 눌림 매수는 1차a/1차b가 모두 지지선 위 양봉 영역(눌림 후 반등 첫 양봉). 박스 돌파 매수는 1차a/1차b가 모두 돌파선 위(박스 안 진입 금지)

**목표가 계산 원칙**
- 목표가 기준: 직전 주요 저항대 (이전 고점, 박스권 상단)
- 평단이 낮아질수록 동일 목표가까지의 **수익률이 자동 상승**
- 체결 단계별로 목표가를 소폭 조정해 **체결 확률 최적화**
- **스윙**: 목표 수익률 5~20% (종목 변동성에 따라)
- **중장기**: 목표 수익률 20~50%+

**손절 라인 선정 (반드시 단일)**
- 마지막 차수 매수가보다 **아래쪽**에 위치, 버퍼 2~5%
- 기법적 근거: 밥그릇 2구간 최저점 / 공구리 라인 / 224일선 이탈
- 손절가는 **기계적으로** 지킨다

**손익비 검증 필수**
- 전량 체결 시 손익비 **1.5:1 이상** 목표
- **2.0:1 이상**이면 이상적

#### 4.6.3 진입가 박스 3요소 통합 표준

확정 진입가 / 조건부 단계 박스는 항상 다음 3요소를 1박스에 통합:

```html
🟣 {단계명} 진입가 {가격}원
⏳ {진입 가능 여부}
└ 필수조건: {필수조건 1개}
```

**진입 가능 여부 (3택 1)**:
- `✅ 진입 OK (조건 충족)` — 즉시 진입
- `⏳ 진입 대기 (조건 미충족)` — 조건 충족 시까지 대기
- `🚫 진입 금지` — 추세 붕괴 등 진입 금지

**필수조건 1개**: 양봉 / 거래량 / 체강 / 갭 / 이평선 등에서 **현재 가장 중요한 단일 미충족 조건**만 표기. 여러 조건 나열 금지.
