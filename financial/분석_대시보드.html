• 2구간 (돌파): 박스 상단 돌파 + 거래량 폭증
• 3구간 (메인 상승): 추세 가속 + 신고가 도전
• 4구간 (분배): 5파 종료 후 ABC 조정 / 매물대 처분
🧱 공구리 기법
(공구리 배너)		 매물대(지지·저항)의 단단함을 콘크리트에 비유. 두꺼운 공구리는 여러 번 다져진 가격대 = 단단한 지지·저항 → 손절 진입·후퇴 금지 기준으로 활용. • 두꺼운 공구리: 박스 5달+ 횡보 / 224일선 / 일목구름 → 종가 이탈 시에만 손절
• 중간 공구리: 60·112일선 / 2~3개월 매물대 → 양봉 첫 지지 확인 진입
• 얇은 공구리: 5·20일선 단기 이탈은 정상 (손절 사유 아님)
운영 룰: ① 두꺼운 공구리 종가 이탈 = 즉시 손절 (추세 무효 · 후퇴 금지) ② 두꺼운 공구리 위 안착 + 양봉 + 거래량 정상 = 진입 신뢰도 ⭐⭐⭐ ③ 밥그릇 4구간 + 공구리 이탈 임박 = 진입 보류 / 손절 검토
